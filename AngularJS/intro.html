Знайомство з Angular JS

Огляд

Angular - сімейство фреймворків для створення UI-орієнтованих програм. Поділяють: Angular JS (або Angular-1) та Angular (або Angular-CLI)

Встановлення

Angular JS встановлюється шляхом підключення скрипту (angular.min.js) який можна завантажити у папку з сайтом для спрощення роботи.

Перевірка роботи (підключення):

  • Angular JS діє (обробляє) лише той блок HTML, який позначений атрибутом ng-app. Частіше за все - це весь документ, хоча можуть бути і менші блоки. Додаємо атрибут до початкового тегу <html lang="en" ng-app>
  • Angular JS підтримує вирази (інструкції, що мають результат). Вираз оформлюється у подвійні фігурні дужки {​{ 2 + 3 }​} = {{ 2 + 3 }} Якщо з'являється результат обчислення, то підключення успішне.

Зв'язування та моделі

Angular впроваджує патерн MVVM, зв'язок забезпечується через концепцію моделей. Моделі - "постачальники даних", введення моделей можливе як через HTML (зазвичай input), так і через код. У HTML моделі позначаються атрибутом ng-model Доступ до даних моделі здійснюється через вирази: {​{ userName }​} = {{ userName }}. Зв'язування може використовуватись саме-по-собі, наприклад, для формування документів.
